# NaFlorestaBuy — Admin Quick Guide

## Create product and enable builder
- Create a **Variable product** in WooCommerce.
- Add attributes/variations and publish/update.
- Open **NaFlorestaBuy** tab and enable builder.
- Select the variations that should be available in the matrix.
- Save product.

## Apply a preset
- In the NaFlorestaBuy product tab, choose an existing preset key.
- Click apply preset action and save product.
- Confirm variation list and schema are updated.

## Use import/export
- Export current config from product tab (JSON).
- Paste/import a valid JSON config.
- Save product and refresh to confirm loaded settings.
- If invalid JSON is provided, correct format and retry.
