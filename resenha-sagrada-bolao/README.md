# Resenha Sagrada Bolão

Atualização pontual para anexar bandeiras aos confrontos informados pelo cliente.

## Auditoria da atualização

- A atualização só usa filtros de renderização (`the_content` e `widget_text_content`).
- Não executa `CREATE`, `INSERT`, `UPDATE`, `DELETE`, `dbDelta`, `update_option` ou alterações em tabelas.
- Não cria shortcode novo.
- Não altera cálculo, armazenamento ou pontuação de palpites.

## Entrega sem binários

O pacote ZIP binário foi removido do repositório. A atualização fica versionada como arquivos-fonte do plugin para manter compatibilidade com revisão e integração por Git.
